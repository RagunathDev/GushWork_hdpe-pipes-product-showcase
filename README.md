# Modern Industrial Website

A fully responsive, pixel-perfect industrial product showcase website built using **Vanilla HTML, CSS, and JavaScript**.

This project focuses on creating a premium modern UI/UX experience with responsive layouts, interactive product galleries, sticky navigation, image zoom functionality, and scalable frontend architecture.

---

## 🔥 Project Overview

This project was developed as a frontend implementation assignment based on a provided Figma design.

The goal was to:
- Replicate the design with pixel-perfect accuracy
- Build a responsive layout for all screen sizes
- Implement smooth user interactions
- Maintain clean and scalable frontend architecture
- Use only Vanilla HTML, CSS, and JavaScript

## 📱 Responsive Breakpoints

| Screen Size | Padding Rules |
|---|---|
| Below 1600px | 100px all sides |
| Below 1440px | 100px all sides |
| Below 1200px | 80px all sides |
| Below 1080px | 80px top/bottom, 60px left/right |
| Below 800px | 80px top/bottom, 48px left/right |
